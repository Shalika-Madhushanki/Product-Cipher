How to use;

*The message which is needed to encrypt should be written in the input text file.
*Then go to "out"->artifacts->Product_Cipher_jar. 
*then There is a executive jar file named Product cipher.Open it double click.
*Enter your preferred Security key in the encrypt text field and submit.
 (Security key can be any integer number with any length)
*Encrypted message will be stored in the encrypted text file.
*In order to decrypt the message correct security key should be entered in 
 the decrypt text field.
 (which is used to encrypt the message)
*Decrypted message will be displayed in a dialogue box as well as the Output text file.
 If the security key is wrong, meaningless message will be displayed.
